# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.40.x   | :white_check_mark: |
| < 1.40.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in HA-AgentHub, please report it privately via the [GitHub Security Advisories](https://github.com/mainzerp/ha-agenthub/security/advisories/new) page.

Please do not open public issues for security bugs. We will respond within 5 business days and work with you to validate, fix, and disclose the issue responsibly.
