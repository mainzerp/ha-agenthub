Filler / Interim / Cancel Responses

When enabled, the assistant speaks a short filler phrase while waiting for slow agents (e.g. web search; only for agents marked as high-latency) and uses the same lightweight LLM to generate a natural spoken acknowledgement when the user dismisses the turn (e.g. "never mind", "forget it").
